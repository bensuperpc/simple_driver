# simple_kernel_driver

A simple kernel driver for Linux.

## Requirements

- Linux kernel headers
- GCC
- Make

## Build

```sh
make
```

## Install (Load)

```sh
sudo make load
```

## Uninstall (Unload)

```sh
sudo make unload
```

## Clean

```sh
make clean
```
